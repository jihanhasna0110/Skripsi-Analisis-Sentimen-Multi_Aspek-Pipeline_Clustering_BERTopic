Tentang Sistem

Bagaimana Sistem Ini Bekerja?

Sistem menggunakan pendekatan pipeline 2 tahap per aspek — pertama mengidentifikasi relevansi ulasan, lalu menentukan sentimen secara akurat.

🩺

Fitur Medis Aplikasi

Analisis ulasan terkait booking dokter, jadwal, telemedicine, dan fitur medis lainnya.

💊

Pelayanan & Kepuasan

Analisis ulasan terkait pelayanan dokter, perawat, administrasi, dan kepuasan pasien.

⚙️

Teknis Aplikasi

Analisis ulasan terkait performa, kecepatan, bug, dan aspek teknis aplikasi.

📈

Output Sentimen

Setiap aspek menghasilkan label Positif, Negatif, atau Netral secara independen.

Alur Analisis

1
Input Ulasan

Masukkan teks ulasan pengguna atau upload file CSV berisi banyak ulasan sekaligus.

2
Preprocessing

Teks diproses melalui stemming (PySastrawi) dan transformasi TF-IDF.

3
Klasifikasi Relevansi

Model SVM menentukan apakah ulasan relevan terhadap setiap aspek.

4
Klasifikasi Sentimen

Untuk ulasan relevan, model SVM menentukan sentimen Positif / Negatif / Netral.

5
Visualisasi Hasil

Hasil ditampilkan dalam tabel, grafik distribusi, dan kesimpulan otomatis.
